StarsMagnet

Django server created as a college project. It allows register companies and give them feedback by users.

Tech Stack

  • Python
  • Django framework
  • Django rest framework
  • MySQL database

other python modules

  • PyJWT
  • mysqlclient
  • django-cors-headers
  • django-jazzmin
  • python-dotenv
  • requests
  • sqlparse

API Reference

Log in by user

POST /api/login

Register new user account

POST /api/register

Log in by company

POST /api/company/login

Register new company

POST /api/company

reset company token

POST /api/token/reset

refresh jwt

POST /api/token/refresh

auto log in

POST /api/login/auto

Get company info

GET /api/company

Add opinion by user

POST /api/opinion

Add response to opinion by company

POST /api/opinion/company

Get opinions for specific company

GET /api/opinion/list/company/<int:company_id>
GET /api/opinion/list/company/<int:company_id>/pageable

Get categories

GET /api/category/pageable/<int:amount>
GET /api/category
GET /api/category/all

Get companies by category

POST /api/category/company
POST /api/category/company/pageable

Get companies by query

POST /api/search
POST /api/company/pageable
